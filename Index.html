Загрузка оценок

  1. Перейдите во вкладку "Загрузка оценок".
  2. Нажмите кнопку "Выберите файл", чтобы выбрать файл для загрузки.
  3. Выберите нужный файл из каталога и нажмите "Загрузить".

После этого данные из файла будут загружены в систему.

Редактирование журнала

  1. Перейдите во вкладку "Создание/редактирование журнала".
  2. Используйте таблицу слева для редактирования или добавления данных об учениках.
  3. Для редактирования нажмите "Редактировать" и внесите изменения.
  4. Для добавления нового ученика заполните поля справа и нажмите "Добавить ученика".
  5. Чтобы удалить ученика, используйте кнопку "Удалить".

Следуя этим шагам, вы сможете управлять данными в журнале.

Статистика (табличная)

Для просмотра статистики по классам и предметам, перейдите в соответствующий раздел.

  • Средняя оценка по каждому предмету.
  • Медиана оценок.
  • Количество учеников с различными оценками.
  • Процентное соотношение оценок по каждому предмету.

Статистика (графическая)

Для визуализации данных по оценкам, перейдите в графический раздел.

  • Графики для оценки средней и медианной оценок.
  • Графики для количества учеников с различными оценками.
  • Процентное соотношение оценок.
